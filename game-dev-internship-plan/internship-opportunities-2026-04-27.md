# 游戏开发实习机会搜索清单

更新时间：2026-04-28

适用对象：大一或低年级本科生，目标是先拿到第一段真实游戏开发、Unity 交互开发、VR/AR 开发、游戏工具或游戏后端相关实习。当前阶段优先级不是“大厂游戏核心项目”，而是能参与真实开发、有导师或 Code Review、能把项目经历写进简历。

## 筛选口径

优先保留：

- 游戏客户端开发实习生、Unity 开发实习生、U3D 实习生、UE 客户端实习生、Gameplay 程序实习生。
- Unity 交互开发、VR/AR 开发、数字孪生/仿真开发、游戏工具开发、游戏服务端实习。
- 学历要求为本科、在校生、学历不限，或明确接受 2027/2028 届及以后。
- 技术要求是 Unity、C#、C++、Lua、Cocos、TypeScript、数据结构、算法、简单图形学、Demo 作品。
- 每周 3 到 5 天、至少 3 到 6 个月，能参与项目开发，能产出可写进简历的模块。

暂时不优先：

- 明确要求大三、大四、2026 届、2027 届且当前年级不匹配的岗位。可以收藏 JD，但不要作为第一批主投。
- 只招游戏测试、游戏运营、游戏客服，但没有开发内容的岗位。
- 明确要求 1 到 3 年商业项目经验、上线手游经验、完整 MMO/SLG 项目经验的岗位。
- 要求 7 天到岗、长期高强度加班、无导师、无实习证明、无薪或只给补贴的岗位。
- 要求交钱、培训贷、押证件、异地封闭培训、先培训再推荐就业的岗位。

## 主流招聘软件使用顺序

### 日常实习

目标是先拿到第一段真实经历。低年级不要只盯腾讯、网易、米哈游、莉莉丝、完美世界、字节游戏等高竞争岗位，可以先从中小游戏团队、VR/AR 公司、交互展示公司、教育游戏、仿真软件、Unity 工具开发切入。

1. BOSS 直聘：主投。适合找中小团队、VR/AR、Unity 交互开发、本地软件公司，沟通效率高。
2. 牛客：主投游戏公司实习、校招实习和面经复盘，适合看 JD 要求和笔试面试范围。
3. 智联招聘：辅助。适合搜本地 Unity、VR、交互、数字孪生、软件开发实习。
4. 实习僧：辅助。适合搜日常实习和长期实习，但下线岗位较多，要重点看刷新时间和截止日期。
5. 公司官网/公众号：后期投目标游戏公司、游戏工作室和暑期实习。

### 暑期或转正实习

目标是拿更正式的游戏客户端实习。通常更看年级、到岗时间、算法笔试、作品和项目经历。

1. 牛客：主投游戏行业专场、校招实习、暑期实习和内推。
2. 公司官网：主投大厂和中大厂游戏业务，如腾讯游戏、网易游戏、米哈游、莉莉丝、完美世界、鹰角、叠纸、巨人、4399、吉比特、三七互娱等。
3. BOSS 直聘：补充投中小公司和可转正岗位。
4. 实习僧：补充搜明确写 `Unity`、`游戏客户端`、`VR/AR` 的岗位。

## UE/Unreal 岗位说明

UE 不是没有机会，而是公开实习岗位比 Unity 少，而且更常藏在 `游戏客户端开发实习生`、`UE4引擎实习生`、`UE5客户端开发`、`Gameplay开发实习生` 这些关键词下面。它通常比 Unity 岗更看重：

- C++ 基础：指针、引用、虚函数、构造析构、智能指针、STL、内存管理。
- Unreal 基础：Actor、Pawn、Character、Controller、Component、UObject、UPROPERTY、UFUNCTION、蓝图和 C++ 协作。
- 计算机基础：数据结构、算法、操作系统、网络、图形学。
- 项目证据：至少一个 UE 小 Demo，最好包含角色控制、相机、动画、UI、碰撞、简单 Gameplay 或工具。
- 到岗和年级：很多 UE 岗要求 26/27 届、每周 4 到 5 天、连续 3 到 6 个月，大一阶段更适合作为中期目标。

如果现在刚开始，建议主线仍然先用 Unity 做出可试玩作品；如果你明确喜欢 UE，就在第 5 到 8 周补一个 UE5 C++ Gameplay 小 Demo，不要只做纯蓝图演示。

## JD 反推学习清单

看到 JD 时，把要求拆成下面几类：

| JD 要求 | 你要准备的证据 |
| --- | --- |
| 熟悉 Unity | 一个可试玩 Demo + README + 视频 |
| 熟悉 C# | 项目代码结构 + C# 八股卡片 |
| 熟悉 C++ | C++ 基础题 + 小型 C++/Unreal Demo 可选 |
| 熟悉 Lua | Lua 表、闭包、元表、热更新概念 |
| 熟悉 Cocos/TS | 一个小游戏 Demo 或 TypeScript 基础项目 |
| 数据结构算法 | Hot 100 基础题 + A* 或寻路 Demo |
| 图形学基础 | 渲染管线、MVP、透明、阴影、Shader 小效果 |
| 性能优化 | Profiler 截图 + GC/Draw Call/资源优化记录 |
| 热爱游戏 | 你玩过什么、分析过什么、你的作品如何改进体验 |
| 团队协作 | Git 提交、Issue、README、代码规范、任务拆分 |

- Git 仓库能打开，README 不空。
