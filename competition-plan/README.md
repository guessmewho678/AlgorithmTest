# 计算机专业竞赛规划

更新时间：2026-05-06

适用对象：福建理工大学计算机相关专业本科生。所有校内报名、组队名额、费用报销和指导教师要求，优先以福建理工大学教务处、创新创业创造教育学院、计算机科学与数学学院发布的通知为准；外校通知只作为时间节奏参考。

## 总体建议

不要所有比赛一起报。对计算机专业来说，最值得长期投入的是四条线：

1. 算法编程线：[CCF CSP](https://www.cspro.org/)、[蓝桥杯](https://dasai.lanqiao.cn/)、[团体程序设计天梯赛](https://gplt.patest.cn/)、[ICPC](https://icpc.global/) / [CCPC](https://ccpc.io/)。适合打基础、准备机试和笔试。
2. 项目创新线：[中国大学生计算机设计大赛](https://jsjds.blcu.edu.cn/)、[服创大赛](https://www.fwwb.org.cn/)、[中国国际大学生创新大赛](https://cy.ncss.cn/)。适合把 Java 后端、AI 应用、数据分析项目做成作品。
3. 数学建模线：[全国大学生数学建模竞赛](https://www.cmathc.org.cn/mcm/tz/418.html)、[美赛 MCM/ICM](https://www.contest.comap.org/undergraduate/contests/mcm/)。适合训练建模、数据处理、论文写作和团队协作。
4. AI/数据线：[中国高校计算机大赛-大数据挑战赛](https://nercbds.tsinghua.edu.cn/bdc.html)、[人工智能创意赛](https://aicontest.baidu.com/)、[AIGC 创新赛](https://aigc.vivo.com.cn/)、[智能交互创新赛](https://hci.oppo.com/)、网络技术类赛事。适合做机器学习、数据挖掘、智能体和 RAG 项目。

低年级建议先用 `CSP + 蓝桥杯 + 天梯赛` 补算法，再用 `4C + 服创` 包装项目；如果能找到数学和写作能力强的队友，再参加全国数模和美赛。

### C4 赛道怎么选

- 只想先稳扎稳打提升算法能力：选 `团体程序设计天梯赛`，同时准备 `CSP/蓝桥杯`。
- 想做 App 或移动端项目：选 `移动应用创新赛`，有大模型能力可以同时看 `AIGC 创新赛`。
- 想做 AI 应用或智能体：优先看 `AIGC 创新赛`、`人工智能创意赛`、`智能交互创新赛`。
- 想做数据挖掘和机器学习竞赛：选 `大数据挑战赛`，它是 2026 年 5 月后仍比较适合开始的 C4 赛道。
- 想走网络安全、云、边缘计算、车联网：准备 `网络技术挑战赛`，但 2026 年报名已过，建议准备下一届。
- 想碰硬件和 AIoT：选 `嵌入式设计赛`，但需要硬件基础、设备和老师支持。

## 推荐优先级

### 第一优先级：低年级打基础

- [CCF CSP](https://www.cspro.org/)：每年多次，可反复刷分，能直接反映算法能力。
- [蓝桥杯](https://dasai.lanqiao.cn/)：适合个人参赛，门槛比 ICPC 低，适合大一大二建立信心。
- [团体程序设计天梯赛](https://gplt.patest.cn/)：校内组队，训练数据结构和基础算法。

### 第二优先级：做出可展示项目

- [中国大学生计算机设计大赛](https://jsjds.blcu.edu.cn/)：把项目做成完整作品，适合 Java 后端、AI 应用、数据平台、可视化。
- [服创大赛](https://www.fwwb.org.cn/)：企业命题和 AI 专项更贴近真实需求，适合做简历项目。
- [中国高校计算机大赛-大数据挑战赛](https://nercbds.tsinghua.edu.cn/bdc.html)：如果你想走 AI/数据方向，这是 2026 年还来得及参加的比赛。

### 第三优先级：建模和论文能力

- [全国大学生数学建模竞赛](https://www.cmathc.org.cn/mcm/tz/418.html)：2026 年还来得及准备，建议尽快找队友。
- [美赛 MCM/ICM](https://www.contest.comap.org/undergraduate/contests/mcm/)：2026 年已结束，准备 2027 年；英文论文压力较大，需要提前练写作模板。

## 备赛方式

### 算法线

- 每周至少 4 天刷题，每次 60 到 90 分钟。
- 训练顺序：语法和模拟、数组字符串、排序和二分、搜索、动态规划、图论、字符串算法。
- 每次 CSP 或蓝桥杯模拟后，整理错题原因：读题、边界、复杂度、实现、调试。

### 项目线

- 不要直接拿视频项目参赛。把项目改成和校园、学习、社团、课程资料、就业服务相关的真实场景。
- 4C 和服创都需要材料完整：项目说明文档、演示视频、源码或 demo、架构图、创新点、测试数据。
- Java 后端项目可以包装成：校园预约系统、社团活动平台、课程资料 RAG 问答、实习信息聚合与推荐系统。

### 建模线

- 队伍建议 3 人：建模推导、编程实现、论文写作；计算机专业可以主攻编程和可视化，但也要懂模型假设。
- 先练全国数模历年题，再练美赛英文表达。
- 常用工具：Python、pandas、numpy、scikit-learn、matplotlib、LaTeX 或 Word 模板。

### AI/数据线

- 先跑通官方 baseline，再做特征工程、模型调参、交叉验证和误差分析。
- 注意比赛数据和代码不要违规公开；提交前保证结果可复现。
- 如果要和简历结合，赛后写一篇复盘：数据处理、模型选择、评价指标、失败尝试、最终改进。
